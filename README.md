## ⚙️ Development Environment

### Prerequisites
```bash
# Check Java version
java -version

# Check Java compiler version
javac -version
```

### Compilation Commands

#### 1. Basic Compilation
```bash
javac <filename>.java
```

#### 2. Running Java Programs
```bash
java <classname>
```

#### 3. Creating JAR Files
```bash
jar cvf <jarname>.jar *.class
```
